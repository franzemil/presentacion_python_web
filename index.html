Aplicaciones

Son paquetes independientes entre se sí, cada uno contiene:

  • models.py // Donde se define la estructura del DB
  • apps.py // Configuracion de la apliacion
  • migrations/ // Migraciones de los modelos
  • views.py // Vistas (Controlador) de la aplicacion
  • *admin.py // Configuracion de los models dentro del admin
  • *task.py // Areas dentro la aplicacions que requerian un procesamiento externo
  • *utils.py // Tareas comunes (helpers)

Se deben registrar dentro del archivo settings.py 


                    								INSTALLED_APPS = [
                    										...
                    								    'django.contrib.staticfiles',
                    								    'nombre_aplicacion'
                    								]

Modelos

Es la representación de la DB mediante objetos, cada modelo equivale a un tabla 

                    							from django.db import models
                    
                    							class Tabla(models.Model):
                    							    atributo = models.CharField(max_lenght=150, null=True)
                    							    int_atribute = models.IntegerField(default=0)
                    							    fecha_registro = models.DateTimeField(auto_now=True)

Vistas

Contiene la lógica de la aplicación, existen dos tipos 

                    							from django.views import View
                    
                    							def vista_funcion(request):
                    							    if request.method == 'GET':
                    							        .....
                    							    if request.method == 'POST':
                    							        .....
                    
                    							class VistaClase(View):
                    							    def get(self, request):
                    							        ....
                    							    def post(self, request):
                    							        ....

Rutas

Establece los puntos de acceso 


                    							from .mis_vistas import vista_funcion, VistaClase
                    							from django.urls import url
                    
                    							urlspatterns = [
                    								url(r'^/ruta_a_function/$', vista_funcion),
                    								url(r'^/ruta_a_clase/$', VistaClase.as_view()),
                    							]
